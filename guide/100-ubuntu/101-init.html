# アップデートとエディタ

これをしないと始まらない。

参考: Ubuntu のアップデート方法 - Qiita (opens new window)

# アップデート確認
$ sudo apt update

# パッケージをアップデートする
$ sudo apt dist-upgrade

# アップデートで必要なくなったパッケージの削除
$ sudo apt autoremove

私の宗派は Vim なのでついでにインストール。
（どうせ設定編集でしか使わないし...矢印つかえるし...）

# エディタのインストール
$ sudo apt install vim
